Comment aplatir un JSON imbriqué — Transformer des données API complexes en CSV / Excel propre en quelques secondes

Publié le 28 avril 2026 par l'équipe DevToolbox — 7 min de lecture

Presque chaque API renvoie du JSON imbriqué. Vous demandez une liste d'utilisateurs, et dans chaque utilisateur se trouve un objet adresse avec rue, ville, code postal ; peut-être un objet préférences, ou un tableau de commandes. Les tableurs et bases de données, eux, attendent des tables plates. C'est là que l'aplatissement du JSON imbriqué devient essentiel.

Que vous ayez besoin de convertir un tableau JSON en CSV, d'exporter du json to excel online, ou de transformer une réponse d'API brute en un CSV formaté qu'Excel peut lire avec les accents corrects, ce guide vous montrera la méthode la plus rapide et la plus privée — sans écrire une seule ligne de code.

Que signifie réellement « aplatir un JSON imbriqué » ?

Cela signifie prendre une structure comme celle-ci : 

[
  {
    "nom": "Alice",
    "adresse": {
      "ville": "Paris",
      "code_postal": "75001"
    }
  }
]

Et la transformer en tableau plat : 

nom,adresse.ville,adresse.code_postal
Alice,Paris,75001

Le sous‑objet adresse disparaît ; ses champs deviennent de nouvelles colonnes avec des en‑têtes en notation par points. C'est la base du json to excel online, et c'est exactement ce que fait notre Convertisseur JSON en CSV automatiquement lorsque vous cochez l'option Aplatir les objets.

Pourquoi aplatir du JSON ? Raisons concrètes

  • Convertir du JSON d'API en CSV pour des rapports métier. Votre backend envoie des données imbriquées ; votre interlocuteur veut un tableau.
  • Préparer des données pour des bases de données. La plupart des outils SQL importent mieux du CSV plat que du JSON profondément imbriqué.
  • Déboguer des jetons JWT. Souvent, la charge utile d'un JWT contient des claims imbriqués. Vous pouvez décoder le JWT, puis aplatir sa charge utile en CSV pour analyse.
  • Partager avec des collègues non techniques. Ils connaissent Excel, pas les chemins JSON.

Utilisations courantes de l'aplatissement JSON

  • Exporter des réponses d'API vers Excel
  • Importer du JSON dans des bases de données SQL
  • Analyser des données JSON dans des tableurs
  • Créer des rapports CSV à partir d'API
  • Préparer du JSON pour des outils de BI

Aplatir du JSON imbriqué en ligne — pas à pas avec notre convertisseur gratuit

Notre convertisseur CSV pour JSON imbriqué ne nécessite aucun téléversement, aucune connexion, ni aucune communication avec un serveur. Tout reste dans votre navigateur. Voici la procédure :

1. Collez votre JSON

Rendez-vous sur le Convertisseur JSON / CSV et collez votre JSON brut dans le panneau de gauche. Si vous l'avez récupéré d'une API, collez la réponse entière — l'outil peut extraire le tableau pour vous.

2. Spécifiez le chemin du tableau (si nécessaire)

Les API enveloppent souvent le véritable tableau dans data, results, ou users. Utilisez le champ Chemin du tableau pour pointer directement vers la liste que vous voulez convertir. Par exemple, si votre réponse ressemble à : 

{
  "status": "ok",
  "data": {
    "users": [ ... ]
  }
}

Tapez simplement data.users. Le convertisseur ignorera les couches d'emballage et travaillera sur le tableau.

3. Activez l'aplatissement des objets

Cochez la case Aplatir les objets. Cela déclenche l'aplatissement de chaque objet imbriqué en colonnes à notation par points. Vous pouvez voir le résultat immédiatement dans l'aperçu du tableau en direct.

4. Convertissez et téléchargez

Cliquez sur Convertir en CSV →. Le CSV généré apparaît dans le panneau de droite. Pour une compatibilité json csv excel utf8, cliquez sur Télécharger CSV (UTF‑8 BOM) — cela ajoute le marqueur invisible qui indique à Excel d'afficher correctement les accents, les umlauts et tous les caractères spéciaux.

Dans l'autre sens : CSV vers JSON formaté

Si vous avez un CSV plat et souhaitez reconstruire un JSON formaté bien indenté, le chemin inverse est tout aussi simple :

  1. Collez votre CSV dans le panneau de droite.
  2. Choisissez le délimiteur (virgule, point‑virgule, tabulation, pipe) — l'outil le détecte automatiquement si vous avez un doute.
  3. Indiquez si la première ligne contient les en‑têtes de colonnes.
  4. Traitez les cellules vides : considérez‑les comme des chaînes vides, null, ou 0.
  5. Cliquez sur ← Convertir en JSON. Le résultat est un tableau JSON bien formaté, prêt à être utilisé dans votre code.

Pourquoi utiliser notre outil plutôt qu'écrire un script ?

De plus, il fonctionne en local — aucune donnée sensible ne quitte votre machine.

Foire Aux Questions

Comment puis‑je aplatir un JSON imbriqué en ligne sans téléverser mes données ?

Utilisez notre convertisseur JSON vers CSV gratuit. Il fonctionne entièrement dans votre navigateur — collez votre JSON, cochez « Aplatir les objets », et téléchargez le CSV. Aucune donnée n'est envoyée à un serveur.

Puis‑je convertir une réponse d'API JSON avec des objets imbriqués en Excel ?

Oui. Collez la réponse de l'API, spécifiez le chemin du tableau si nécessaire, activez l'aplatissement, puis téléchargez au format CSV avec BOM UTF‑8 pour une compatibilité parfaite avec Excel.

Et si mon tableau JSON est imbriqué sous data.users ?

Utilisez le champ Chemin du tableau pour saisir data.users (ou n'importe quel chemin en notation par points), et le convertisseur extraira et aplatira uniquement ce tableau.

L'aplatissement fonctionne‑t‑il pour des objets profondément imbriqués (3 niveaux ou plus) ?

Oui, l'aplatisseur descend récursivement dans les sous‑objets et concatène les clés avec un point. Il n'y a pas de limite de profondeur.
