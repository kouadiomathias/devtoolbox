Des outils pensés pour le quotidien

DevToolbox regroupe les outils que les développeurs recherchent chaque jour : formater un JSON mal indenté, décoder un token JWT pour débugger une authentification, tester une regex sans écrire de script, convertir un timestamp Unix en date lisible.

Chaque outil fonctionne entièrement dans le navigateur. Aucun appel réseau, aucun backend. Vos données sensibles — tokens d'authentification, données de production, expressions régulières métier — restent strictement sur votre machine.

Outils disponibles

  • JSON Formatter & Validator — Formatage, validation et minification JSON
  • JWT Decoder — Inspection de JSON Web Tokens et vérification d'expiration
  • Regex Tester — Testeur d'expressions régulières avec explication en langage naturel
  • Base64 Encoder / Decoder — Encodage et décodage Base64 et Base64url
  • Unix Timestamp Converter — Conversion de timestamps Unix avec support timezone
  • URL Encoder / Decoder — Encodage et décodage d'URLs (percent-encoding)
