Comparateur de texte avancé

Cet outil vous permet de visualiser les différences entre deux versions d'un texte ou d'un code. Il fonctionne entièrement dans votre navigateur, aucune donnée n'est envoyée à un serveur.

Foire Aux Questions

Diff unifié (unified) vs vue fractionnée (split) : quelle différence ?

La vue split affiche les deux textes côte à côte, avec les différences alignées. La vue unifiée (comme `git diff`) montre un seul flux avec les lignes ajoutées (précédées de `+`) et supprimées (`-`).

Comment fonctionne la comparaison de JSON ?

L'outil peut comparer du JSON brut en mode texte, mais il offre aussi une coloration syntaxique pour repérer facilement les structures. Pour une comparaison sémantique, formatez d'abord votre JSON avec notre formateur JSON.

Puis-je partager mon diff ?

Oui, le bouton "Partager" génère une URL avec l'état complet (les deux textes et options) encodé dans le fragment d'URL. Vous pouvez copier ce lien et l'envoyer à un collègue — tout est décodé localement.

Le diff mot par mot est-il disponible ?

Oui, dans les lignes modifiées, les mots ajoutés ou supprimés sont surlignés en vert et rouge pour une précision maximale.

Quels langages sont supportés pour la coloration ?

JSON, SQL, YAML, JavaScript, Python, et bien d'autres. L'auto-détection essaie de deviner le format. Vous pouvez aussi forcer un langage.

Puis-je comparer de gros fichiers ?

Oui, mais les performances dépendent de votre navigateur. Pour de très gros fichiers, il est recommandé de les diviser.

Cet outil fonctionne-t-il hors ligne ?

Oui, tout le traitement se fait dans votre navigateur. Aucune donnée n’est envoyée à un serveur.
