Questions fréquentes

Qu'est-ce qu'un clip-path en CSS ?

La propriété clip-path découpe un élément HTML selon une forme géométrique. Cet outil permet de faire glisser les points directement sur l'aperçu pour un contrôle visuel total.

Comment sont générés les blobs ?

Les blobs utilisent un algorithme de bruit de Perlin/Simplex pour créer des contours organiques naturels. Ajustez la complexité et la graine (seed) pour des formes uniques.

Le simulateur de daltonisme est-il réaliste ?

Oui, il applique des filtres SVG basés sur le modèle de Brettel-Viénot avec des matrices LMS pour simuler fidèlement la protanopie, deutéranopie et tritanopie.

Puis-je exporter pour Tailwind ou React ?

Absolument. Les boutons dédiés génèrent le code Tailwind CSS (avec syntaxe [clip-path:...]) et JSX pour React. Le code CSS natif est toujours disponible.

Comment partager ma forme ?

Cliquez sur « Partager URL » : tous les paramètres sont encodés dans le hash de l'URL. Envoyez le lien, votre collègue retrouvera exactement la même configuration.

L'outil est-il gratuit ?

Oui, 100% gratuit, sans inscription et sans publicité. Toutes les formes créées vous appartiennent.
