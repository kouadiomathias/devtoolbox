Tools built for everyday use

DevToolbox brings together the tools developers search for every day: formatting JSON, decoding JWT tokens, testing regex, converting Unix timestamps.

Every tool runs entirely in the browser. No network calls, no backend. Your sensitive data — auth tokens, production data, business regex — stays strictly on your machine.

Available tools

  • JSON Formatter & Validator — Format, validate, and minify JSON data
  • JWT Decoder — Inspect JSON Web Tokens and verify expiration
  • Regex Tester — Test regex with natural language explanation
  • Base64 Encoder / Decoder — Encode and decode Base64 and Base64url
  • Unix Timestamp Converter — Convert Unix timestamps with timezone support
  • URL Encoder / Decoder — Encode and decode URLs using percent-encoding
