Convert JSON to CSV Online: Flatten Nested Objects, Export Excel‑Ready Files (No Server Upload)

Published on April 24, 2025 · Updated April 28, 2025 by the DevToolbox Team — 8 min read

Need to convert JSON to CSV quickly, without uploading your data anywhere? Whether you pulled an API response and need to open it in Excel, or you have a nested JSON structure that must become a flat table, this guide walks you through the fastest, most private way to do it. We’ll also touch on how to decode JWT tokens when they’re part of your JSON payloads.

Why you need a JSON to CSV converter

JSON is the backbone of modern APIs, but spreadsheets and databases still speak CSV. Converting JSON into CSV lets you:

  • Open API responses directly in Excel or Google Sheets
  • Flatten nested objects into flat columns (e.g., address.city)
  • Prepare data for ETL pipelines or database imports
  • Share data with non‑technical colleagues who only work with tables

But there’s a catch: many online converters require you to upload your file to a server. That’s not acceptable when you handle sensitive customer data, financial records, or authentication tokens. Our tool does everything inside your browser — zero server upload.

How to convert from JSON to CSV in 5 steps

Step 1: Paste or drag & drop your JSON

Open the JSON / CSV Converter. You can paste raw JSON into the left panel, or drag a .json file from your computer. The interface detects the format automatically: if you paste JSON, it prepares a CSV conversion.

Step 2: Extract the target array (path)

APIs rarely return a plain array; they wrap it inside data, results, or users. Look at this response: 

{
  "success": true,
  "data": {
    "users": [
      {"id": 1, "name": "Alice", "age": 30},
      {"id": 2, "name": "Bob", "age": 25}
    ]
  }
}

To convert JSON to CSV from this, enter data.users in the Array path field. The tool will skip the wrapper and only process the array you need.

Step 3: Flatten nested objects

What about embedded objects like an address? 

[
  {
    "name": "Alice",
    "address": {
      "city": "Paris",
      "zip": "75001"
    }
  }
]

Without flattening, the address column would contain [object Object] — useless. Check the Flatten objects box, and you’ll get clean columns like address.city and address.zip.

Step 4: Convert and preview the table

Click Convert to CSV →. A live table preview appears, showing row count, columns, and the converted data. You can immediately spot if something went wrong (for instance, missing values or incorrect nesting) and adjust the options.

Step 5: Download Excel‑ready CSV (UTF‑8 BOM)

Click Download CSV (UTF‑8 BOM) to get a file that opens directly in Excel without character corruption. The UTF‑8 BOM tells Excel that the file contains accents, special letters, or emojis — essential for any non‑English dataset.

Converting from CSV to JSON

The reverse process is just as straightforward:

  1. Paste your CSV into the right panel.
  2. Pick the delimiter (comma, semicolon, tab, or pipe). The tool auto‑detects it if you’re not sure.
  3. Specify whether the first row contains headers. If not, columns will be named col1, col2, etc.
  4. Choose how to handle empty cells: as null, empty string, or 0.
  5. Click ← Convert to JSON to get a formatted, reusable JSON.

When you also need to decode a JWT token

While working with JSON, you might encounter JSON Web Tokens (JWT) inside your data — for example, a user session token returned by an API. If you’ve ever searched for jsonwebtoken decode, you’ll want our dedicated JWT Decoder. It parses the header, payload, and signature of any JWT without leaving your browser. Combined with the JSON/CSV converter, you can extract a token from an API response, decode it, and then flatten the payload into CSV columns — a real time‑saver for debugging auth flows.

Pro tips for accurate conversions

  • Validate your JSON first: use the JSON Formatter to check syntax and indentation.
  • Preserve leading zeros: zip codes like 01234 can lose their zero when interpreted as numbers. Wrap them in quotes in the CSV.
  • Large files: the tool processes everything locally, but for files over 10 MB, consider a command‑line script.
  • Excel French locale: if you often convert json format to csv for French Excel, select semicolon as the delimiter to match the default separator.
