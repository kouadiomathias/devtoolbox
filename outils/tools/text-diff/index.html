Frequently Asked Questions

What is a text diff tool?

A text diff tool compares two versions of text and highlights the differences. It is commonly used by developers to compare code changes, but it can also be used for documents, JSON, or any plain text.

Unified diff vs split view: what's the difference?

Split view displays both texts side by side with aligned differences. Unified view (similar to git diff) shows a single combined view where additions are marked with + and deletions with -.

How does JSON comparison work?

The tool compares JSON as text but also provides syntax highlighting for better readability. For more accurate results, format your JSON first using our JSON formatter.

Can I compare code with this tool?

Yes, you can compare code written in JavaScript, Python, SQL, JSON, YAML, and more. Syntax highlighting helps you quickly identify differences in structure and logic.

Is word-by-word comparison available?

Yes, within modified lines, individual words are highlighted. Added words appear in green and removed words in red for precise comparison.

Can I compare large files?

Yes, but performance depends on your browser and device memory. For very large files, it is recommended to split the content into smaller parts.

Can I share my diff results?

Yes, the "Share" feature generates a URL containing the full comparison (texts and options). This link can be shared and opened directly without sending data to a server.

Which languages are supported for syntax highlighting?

The tool supports JSON, SQL, YAML, JavaScript, Python, and many other languages. Auto-detection is enabled, but you can manually select a language if needed.

Is my data secure?

Yes, all processing is done locally in your browser. No data is sent to any server, ensuring complete privacy.
