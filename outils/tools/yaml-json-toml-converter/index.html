Frequently Asked Questions

Why a different YAML/JSON/TOML converter?

Unlike basic tools, this one preserves types (dates remain strings if desired), handles YAML anchors, validates input rigorously and provides fine control over the output format.

How to enable strict typing?

Check “Strict typing (dates → string)” in the YAML options. Dates like 2024-01-01 will be kept as strings rather than being converted to JavaScript Date objects.

What is semantic comparison?

Two YAML documents can be syntactically different but represent the same structure. Our comparator parses both, normalises key order and checks deep equality.

Is my data sent to a server?

No. All processing (parsing, conversion, comparison) runs inside your browser. No data leaves your machine.
